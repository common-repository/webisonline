== Description ==
Онлайн-консультант WebisOnline позволяет Вам общаться в чате с посетителями вашего сайта.

WebisOnline поможет вам рассказать о всех преимуществах вашей компании.
Вы узнаете, чего ожидают ваши клиенты, и сможете более качественно показать свои предложения, увеличить конверсию и продажи. 

== Installation ==
Укажите ваш email, пароль и адрес сайта, на котором вы хотите использовать онлайн-консультанта. Мы автоматически создадим для вас учётную запись в сервисе WebisOnline. После этого у вас на сайте сразу появится виджет сервиса (интерактивный ярлычок), при помощи которого ваши посетители смогут с вами связаться. Для того, чтобы принимать запросы и общаться с посетителями, можно использовать личный кабинет на сайте webisonline.ru или отдельную программу, которую можно скачать в личном кабинете.
В личном кабинете вы можете настроить внешний вид виджета и автоматические приглашения, смотреть статистику, настраивать свой тариф и отправлять запросы в техподдержку.
После установки модуля необходимый код автоматически запишется в шаблон вашего сайта, там самым вам нужно будет настроить только внешний вид через личный кабинет.
